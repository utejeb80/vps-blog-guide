## 个人博客用VPS到底需要什么？先搞清楚这个定义

个人博客用的 VPS，核心诉求只有三条：**速度够快、稳定不挂机、价格不心疼**。

速度够快，说的不是服务器算力，而是网络线路质量——你的读者从中国大陆打开你博客的那一秒，延迟决定他们会不会等下去。稳定不挂机，说的是宕机了你能不能快速恢复，或者干脆商家能否保证在线率。价格不心疼，个人博客流量不大，每年花个三五百块在 VPS 上完全合理，但也没必要为一个小博客付企业级账单。

把这三条对应到选购参数：**线路 > 稳定性 > 配置 > 价格**。

很多新手反过来看，觉得配置越大越好，然后买了个国内流量便宜、延迟奇高的机器，结果 Google 收录慢、读者跳出率高。

**Q：个人博客需要多大内存的VPS？**

**Q：CN2 GIA-E 和 CN2 GT 有什么区别？**
